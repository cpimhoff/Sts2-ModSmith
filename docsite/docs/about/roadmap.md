# Roadmap

ModSmith is a living project, and Slay the Spire 2's modding scene is in its infancy in general.

Currently there's still some content that ModSmith does not support and requires custom patching:
- New characters
- New base pools
- New encounters
- New bosses
- New acts
- New modifiers
- New card enchantments

Future versions will add native support for these!
