# Example events

## The Gold Coin Room

A multi-page event where the player finds a gold coin and must decide whether to pocket it or gamble by flipping it — with the option to keep doubling on successive heads. Demonstrates multi-page event state machines, branching choices, RNG, and dynamic localization variables.
