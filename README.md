# Slay the Spire 2 ~ ModSmith

Base mod for adding content to Slay the Spire 2.

This mod provides shared logic which other mods may build off of, in a way that allows for multiple mods to contribute logic without conflicting with one another. You may see this mod in the "dependencies" or "required" section of other mods.

See documentation at https://cpimhoff.github.io/Sts2-ModSmith/
