Remarks

Localization

relics.json must contain localization keys for the relic:

[ID].title       // The name of the relic.
[ID].description // The body text of the relic, describing its effects.
[ID].flavor      // Flavor text for the relic.

The path to a larger version of the icon for the relic. This defaults to the same value as PackedIconPath.

The path to the icon for the relic. Most relics are around (85px x 85px).
