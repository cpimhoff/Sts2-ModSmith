Base class for mod-defined powers.

Note that "power" does not exclusively refer to "Power" cards in the game. Powers are any status effect applied to a creature during combat, including debuffs like "Vulnerable".

Remarks

Localization

powers.json must contain localization keys for the power:

[ID].title            // The name of the power.
[ID].description      // The body text of the power when it is not applied to a target. Used for tooltips in cards and potions.
[ID].smartDescription // The body text of the power when it is applied to a target. Can reference dynamic variables.

Properties

BigIconPath

The path to a larger version of the icon for the power. This defaults to the same value as PackedIconPath.

protected virtual string BigIconPath { get; }

Property Value

string

PackedIconPath

The path to the icon for the power. Most powers are around (64px x 64px).

If not provided, a default image will be used.

protected virtual string PackedIconPath { get; }

Property Value

string

StackType

How the power stacks.

  • PowerStackType.Counter: The power can stack, summing the Amount of the power, and potentially increasing its effect or duration.
  • PowerStackType.Single: The power does not stack. Gaining multiple instances of the power does nothing.
public override abstract PowerStackType StackType { get; }

Property Value

PowerStackType

Type

Whether the power is a buff or debuff. Used by artifact and similar effects to determine whether to negate or cleanse a power.

public override abstract PowerType Type { get; }
