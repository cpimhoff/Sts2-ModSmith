Remarks

Localization

potions.json must contain localization keys for the potion:

[ID].title       // The name of the potion.
[ID].description // The body text of the potion, describing its effects.

Usage

When the potion can be used.

  • CombatOnly: The potion is only usable by the player during combat.
  • AnyTime: The potion can be used any time by the player.
  • Automatic: Not used directly by the player; instead the potion triggers itself. For example, FairyInABottle is used automatically when the player would die.
  • None: The potion cannot be used. 
public override abstract PotionUsage Usage { get; }
