Install ModSmith

Note

MegaCrit has not yet implemented Steam Workshop support for Slay the Spire 2. ModSmith will be much simpler to install once that system rolls out.

ModSmith is installed like any other mod. First, download the latest version of the mod.

Then unzip and drag the ModSmith folder (the one containing the ModSmith.json file), into your Slay the Spire 2 mods/ directory (create one if it doesn't exist).

Platform Mod folder location
Windows C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\mods
Linux ~/.local/share/Steam/steamapps/common/Slay the Spire 2/mods
macOS ~/Library/Application Support/Steam/steamapps/common/Slay the Spire 2/SlayTheSpire2.app/Contents/MacOS/mods

If you are developing mods, move on to set up your development environment.
